transition-timing-function is used to affect the speed of transition from start to end

linear-this div wil have same transition speed from start to end

ease-This div will have the same transition speed from slow start and fast in middle to a slow end

ease-in-This div will have the same transition speed from slow start

ease-in-This div will have the same transition speed from slow end

ease-in-This div will have the same transition speed from slow start and slow end
